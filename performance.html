Analyzing performance

Audits

  • Audits panel > Run Audits

Code coverage

  • Command Menu > Show Coverage
  • Drawer Menu > Coverage

Performance panel

  • Performance panel > Start Profiling And Reload Page
  • Keyboard controls:
    • W to zoom in
    • S to zoom out
    • A to pan left
    • D to pan right
